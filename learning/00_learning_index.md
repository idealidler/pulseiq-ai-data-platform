# PulseIQ Learning Pack

This learning pack is designed for NotebookLM and is based only on the architecture and operations that exist in this repository.

Use these files in order:

1. `01_project_overview.md`
2. `02_data_generation_and_ingestion.md`
3. `03_dbt_transformations_and_serving_models.md`
4. `04_semantic_layer_and_schema_context.md`
5. `05_embeddings_and_retrieval.md`
6. `06_chat_orchestration_and_answering.md`
7. `07_evaluation_and_quality.md`
8. `08_frontend_and_user_experience.md`
9. `09_end_to_end_request_lifecycle.md`
10. `10_what_to_learn_next.md`

How to use this pack:

- Start with the overview to understand the whole system.
- Study one chapter at a time.
- After each chapter, open the referenced project files and compare the explanation with the code.
- Use NotebookLM to ask follow-up questions like:
  - "Explain this chapter as if I am new to data engineering."
  - "What tradeoffs were made here?"
  - "Which part of the repo implements this?"
